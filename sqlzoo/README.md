# SQLZoo Solutions

My solutions to the [SQLZoo](https://sqlzoo.net/wiki/SQL_Tutorial) tutorial exercises - a free, browser-based SQL learning resource that gives immediate feedback on queries.

Each file corresponds to one tutorial section, with each problem commented to show what was being solved.

## Files

- [`01_select_basics.sql`](./01_select_basics.sql) — SELECT basics
- [`02_select_from_world.sql`](./02_select_from_world.sql) — SELECT from World
- [`03_select_from_nobel.sql`](./03_select_from_nobel.sql) — SELECT from Nobel
- [`04_select_within_select.sql`](./04_select_within_select.sql) — SELECT within SELECT
- *(more added as I complete them)*

## A note on SQL dialect

SQLZoo runs MySQL / MariaDB, which differs slightly from SQLite (used in my [CO2 emissions project](../co2-emissions)) and from PostgreSQL. Working across
dialects builds genuine SQL fluency.

## Why these are here

SQLZoo is excellent for building SQL fluency through immediate feedback. I'm working through it as part of strengthening my data analysis skills - for applied work, see the [`co2-emissions/`](../co2-emissions) project.
